General problems

  • Hardcoded dependency on framework
  • Lack of documentation/comments
  • Hardcoded coupling between the test-runner UI and SPT
  • Eclipse dependencies in used libraries

Sunshine's problems

  • No custom parser registration
  • Single threaded (no parse locks required though)
  • No way to obtain resolvers (it does now)
  • No analysis of a single file
    (I'm afraid multi-file analysis would break the dynamic rules)
